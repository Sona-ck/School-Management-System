# School-Management-System
Full-featured School Management System built with Django, featuring student, teacher, and parent dashboards, attendance tracking, exams, scholarships, and transport fee management.
## 🚀 Features
- Student Management
- Teacher Dashboard
- Parent Dashboard
- Attendance Tracking
- Exam & Results Management
- Scholarship System
- Transport Fee Management

## 🛠️ Tech Stack
- Python
- Django
- SQLite
- HTML, CSS, Bootstrap

## ⚙️ Installation

```bash
git clone https://github.com/Sona-ck/School-Management-System.git
cd school-management-system
pip install -r requirements.txt
python manage.py runserver
